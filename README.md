# BannerPrint
Prints banner with or without the border

```
banner = Banner(9, 9, border=True)
foo = ''' ####
#####
  ###
  ###
  ###
#######
#######
'''
banner.insert(foo,1,1)
banner.get()
```
```
+---------+
|         |
|  ####   |
| #####   |
|   ###   |
|   ###   |
|   ###   |
| ####### |
| ####### |
|         |
+---------+
```
